# video/audio api

1. 播放与暂停

- 属性：paused    是否停止

- 方法：play()      开始播放；pause()     暂停

2. 时间

- 属性：duration    总时间    currentTime 当前时间

- 监听事件 timeupdate

3. 媒体相关

- playbackRate      速率

- volume            声音

- requestFullscreen()   可以设置document.documentElement全屏

再设置video及其容器宽高为window.screen
