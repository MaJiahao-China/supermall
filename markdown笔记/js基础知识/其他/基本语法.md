# 语句

1. 行(line): js程序的执行单位。一般，一行就是一个语句。

2. 语句(statement): 为了某种任务而进行的操作。语句以;结尾，一个;就表示一个语句结束。

3. 表达式：一个为了得到返回值的计算式。**表达式不需要分号结尾。**

4. 语句和表达式的区别：**前者主要为了进行某种操作，一般情况下不需要返回值；后者则是为了得到返回值，一定会返回一个值。**

5. 空语句：分号前面可以没有任何内容。

# 变量

## 概念

1. 变量: 对值的具名引用.变量就是为“值”起名，然后引用这个名字，就等同于引用这个值。

如果使用var重新声明一个已经存在的变量，是无效的。

```javascript
    var x = 1;
    var x;  
    x// 1

    //第二次声明的时候还进行了赋值，则会覆盖掉前面的值。
    var x = 1;
    var x = 2;
    //相当于
    var x = 1;
    var x;//无效
    x = 2;
```

# 标识符

标识符（identifier）指的是用来识别各种值的合法名称。如变量名，函数名。

# 区块

区块(block): 用{}将多个相关语句组合在一起。

区块往往用来构成其他更复杂的语法结构，比如for、if、while、function等。

# 条件语句

 if结构，if...else, switch...case, 三元运算符?:

if结构先判断一个表达式的布尔值，然后根据布尔值的真伪，执行不同的语句。

# 循环结构

while, for, do...while, 

break语句用于跳出代码块或循环;终止本轮循环，开始下一轮循环。

# label标签

相当于定位符，用于跳转到程序的任意位置。格式label：语句

3. 两个复合类型（对象、数组、函数）的数据比较时，不是比较它们的值是否相等，而是比较它们是否指向**同一个地址**。

对于两个对象的比较，严格相等运算符比较的是地址，而大于或小于运算符比较的是值。
