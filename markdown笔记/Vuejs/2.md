# ref引用


ref 给一个dom一个属性，存储在属性$refs里；$refs是一个对象，属性是ref赋的值，值是dom对象。

```javascript
    const vm = new Vue({
        el: '#app',
        components: {
            myText: {
                data() {
                    return {
                        colors: ['red', 'green', 'blue'],
                    }
                },
                methods: {
                    change(color) {
                        this.$refs.box.style.backgroundColor = color;
                        console.log(this.$refs);
                    }
                },
                template:`
                <div>
                    <button 
                        v-for="color in colors" 
                        :style='{backgroundColor:color}'
                        @click='change(color)'
                    >{{ color }}</button>
                    <div ref="box" class="box" :style></div>
                </div>
                `,
            }
        }
    });
```

1. 单个dom设置ref引用，返回该对象；

2. 多个dom设置相同的引用，会发生覆盖，只获取最后一个dom

3. v-for创建的dom用ref，返回一个数组，数组里是创建的多个dom
